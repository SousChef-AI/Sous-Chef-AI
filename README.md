# Sous-Chef AI

## Overview
This project is a voice-activated cooking assistant that helps users follow reliable recipes hands-free.  
Unlike traditional AI recipe apps, it uses verified human-created recipes to ensure safety and accuracy.  

## How To Run
- Ensure NodeJS is installed
- "npm install" in CMD
- Download the repo as a ZIP
- Extract ZIP into folder
- Open terminal, "cd" into the location of that folder and run "npm run dev"
- Project should open on localhost in a browser tab

## Key Features
- Voice-controlled step-by-step cooking guidance
- Voice control is activate by speaking "Hey Sous Chef", after which the AI will start listening to your request
- Hands-free timers and instructions  
- Ingredient quantity adjustments  
- Nutritional breakdowns macros, calories, etc.
- Nutritional goals can be set and adjusted
- Nutritional history (diary)  
- Images of finished dishes for reference
- Manages inventory of ingredients and their expiration dates
- Filters for dietary restrictions and prefernces (vegan, allergies, etc.)
